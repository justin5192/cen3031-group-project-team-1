## Overview

CarbonTracker is a Python-based application that enables users to track carbon-emitting activities, set personal reduction goals, and monitor progress. As of Sprint 3, the system includes data aggregation and export features in addition to core activity logging and goal tracking.

This tool is built using Python and the Tkinter GUI library, with JSON used for local data storage.

---

## How to Run

### Requirements
- Python 3.10 or later
- No external libraries required (uses only built-in Python modules)

### Execution

From the project root directory:

```bash
python -m presentation.ui_app
